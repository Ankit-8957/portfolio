# Public Assets Folder

This folder contains public assets that are served directly at the root of the site (e.g. `http://localhost:5173/`).

## Resume Download
We have set up the "Download Resume" button on the portfolio to download the resume from `/Resume.pdf`.

### How to add your resume:
1. Save your resume as a PDF.
2. Rename the file to `Resume.pdf` (capital R).
3. Place `Resume.pdf` in this `public` directory (replacing the placeholder file).

## Deloitte Certification Download
We have set up the "Download Certificate" button in the Certifications component to download your certificate from `/DeloiteCertificate.pdf`.

### How to add your certificate:
1. Save your certificate as a PDF.
2. Rename the file to `DeloiteCertificate.pdf` (single 'l').
3. Place `DeloiteCertificate.pdf` in this `public` directory (replacing the placeholder file).

## Profile Photo
The "About me" section renders your profile photo from `/Ankit.jpeg`.

### How to add your photo:
1. Crop your photo to a square (optional, but recommended).
2. Rename it to `Ankit.jpeg`.
3. Save it in this `public` directory.

If `Ankit.jpeg` is not present, it automatically falls back to showing a placeholder with your initial ("A").
